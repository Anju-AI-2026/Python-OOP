# 👨‍💼 Employee Management System (Python OOP)

## 📌 Project Overview

This is a simple **Object-Oriented Programming (OOP)** project in Python that simulates an Employee Management System. It allows users to store employee details, calculate annual salary, determine bonus, and classify employee level based on salary.

---

## 🧠 Concepts Used

* Classes and Objects
* Constructors (`__init__`)
* Methods
* Conditional Statements
* Loops
* Input Validation
* Menu-driven Program

---

## 📂 Project Structure

```
employee_class/
│
├── employee.py   # Contains Employee class and all methods
├── main.py       # Handles user input and menu system
├── README.md     # Project documentation
```

---

## 🚀 Features

* Add employee details (name, ID, salary, department, address)
* Display employee information
* Calculate annual salary
* Calculate yearly bonus
* Determine employee category (Junior / Mid-Level / Senior)
* Input validation for name, ID, and salary
* Interactive menu system

---

## 🧮 Bonus Logic

* Salary ≤ 40,000 → 20% bonus
* 40,000 < Salary < 120,000 → 15% bonus
* Salary ≥ 120,000 → 10% bonus

Bonus is calculated based on the **annual salary**.
